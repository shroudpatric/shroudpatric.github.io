安卓生成空 AAR 包的步骤

  • 首先打开 Android Studio 新建一个 Empty Activity.
  • Name 随便写一个，Package Name 是包名，全部小写。最小的 API 等级这里需要注意，因为这个包最后要导入 Unity 中，Unity 的平台也是安卓平台，在 Unity 的 Project Settings 中有一个最小 API 等级，在 Android Studio 选择的最小
    API 要小于 Unity 的 Project Settings 的最小等级。

  • 在这里需要导入一个 Jar 包，名称叫做 “class.jar”,在你的 Unity 安装目录下”\Editor\Data\PlaybackEngines\AndroidPlayer\Variations\mono\Release\Classes”。
  • 这个 Jar 包直接复制到刚才新建的 Library 的 libs 中，然后右键这个 Jar 包 选择 Add As Library。(注意截图中 Android Studio 工程的视图)
